=head1 NAME

PW-Gomoku - Gomoku game in ncurses

=head1 SYNOPSIS

B<gomoku>

=head1 DESCRIPTION

ncurses based Gomoku implementation. There is no AI so you have to play against an another human.

=head2 Controls:

Arrows + Enter or mouse + left click
